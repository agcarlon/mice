Installation
============

From PyPI
---------

.. code-block:: bash

   pip install mice

From Source (development mode)
------------------------------

.. code-block:: bash

   git clone https://github.com/agcarlon/mice.git
   cd mice
   pip install -e .

Optional dependencies
---------------------

For experiment scripts, install the optional ``experiments`` dependencies:

.. code-block:: bash

   pip install -e .[experiments]
